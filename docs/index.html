claude-council

Pressure-test high-stakes decisions with a 5-advisor LLM Council for Claude Code

claude-council is an open-source, multi-agent decision framework that runs entirely inside Claude Code. One slash command runs your question through five structured thinking lenses, anonymized peer review, a forced adversarial debate when consensus is suspiciously clean, and dual-chairman synthesis that preserves minority dissent.

Install

npx skills add TorpedoD/claude-council

Requires Claude Code (Pro plan or higher) and jq (brew install jq) for journal features.

Usage

/claude-council I'm torn between staying at my job or joining a startup as cofounder

Force a mode by appending quick or deep to your prompt.

Three Adaptive Modes

ModeAdvisorsPeer ReviewDebateTime
Quick3NoNo~30s
Standard5Yes (5)If consensus ≥ 4/5~90s
Deep5 (expanded)YesAlways + Decision Science~150s

How It Works

1. Triage — reject factual or obvious questions
2. Journal lookup — find prior runs on similar questions
3. Frame + Bias audit — workspace scan, cognitive bias detection
4. Mode selection — Quick / Standard / Deep
5. Fan-out — 3–5 parallel advisor agents
6. Decision Science — structured comparison (Deep mode)
7. Peer review — 5 anonymized reviewers + forced debate
8. Dual chairman — consensus + dissent synthesis
9. Generate outputs — HTML report + markdown transcript
10. Journal append — JSONL write + outcome tracking

The Five Advisors

AdvisorRoleMethod
Red TeamAdversarial attackPre-mortem: assumes failure, traces root cause
First PrinciplesAssumption busterTree-of-Thoughts: 3 reframings from ground up
ExpansionistOption multiplierFinds unlisted options, scores upside/effort
OutsiderCross-domain wisdomChannels an unrelated field's methodology
ExecutorFeasibility checkOODA stage + RICE scoring + data audit

Output

Each run produces two files in the current directory:

  • council-report-{ts}-q{sha}.html — visual report with advisor cards, agreement grid, RICE table, debate transcript, and confidence header
  • council-transcript-{ts}-q{sha}.md — full step-by-step transcript with anonymization map
